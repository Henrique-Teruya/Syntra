# Syntra
O syntra é um SGBD (Sistema de Banco de Dados) de baixo custo de investimento destinado a pequenas empresas, servindo como um inventário de estoque, auxilia o gerenciamento da mesma.

Feito para empresas que querem uma opção de baixo custo, aonde possam aplicar um bom gerenciamento de estoque e demandas seguindo a metodologia "just in time", o Syntra é o SGBD certo para você'
